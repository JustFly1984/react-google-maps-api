# Contributing

![logo](https://raw.githubusercontent.com/JustFly1984/react-google-maps-api/master/logo.png)

We are welcome to any pull request, which has a clear goal and description. Also please use oxlint and oxfmt while developing.

List of contributors:

Alexey Lyakhov <justfly1984@gmail.com>

1. Clone repository
2. git checkout development
3. make your own branch and make PR to development branch after finish with changes.
4. Use `bun install` to install in root folder and in each package directory.

Inside root folder you will find packages:
